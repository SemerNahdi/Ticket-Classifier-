# Ticket Agent Workflow

An automated system for classifying and processing support tickets using AI agents.

## Project Structure

```text
Ticket Agent/
📂 data/
  📂 raw/              # Immutable original datasets
  📂 processed/        # Cleaned/Intermediary data for AI
  📂 output/           # Results generated by the agents
📂 src/                # Source code
  📄 api.py            # FastAPI Backend API
  📂 agents/           # CrewAI/Agent logic
  📂 preprocessing/    # Data cleaning & prep logic
📂 tests/              # Integration and unit tests
📄 .env.example        # Template for API keys
📄 requirements.txt    # Project dependencies
📄 README.md           # Documentation
```

## Setup & Running (Windows)

Follow these steps to set up your environment and run the project:

### 1. Create and Activate Virtual Environment
Open PowerShell or Command Prompt in the project root and run:
```powershell
# Create the virtual environment
python -m venv venv

# Activate the environment
.\venv\Scripts\activate
```

### 2. Install Dependencies
```powershell
pip install -r requirements.txt
```

### 3. Configuration
Copy the environment template and fill in your API keys:
```powershell
copy .env.example .env
```

## Usage

Once the environment is active, you can run the following modules:

### Data Preprocessing
Cleans the raw Excel data and prepares JSON datasets for the AI.
```powershell
python src/preprocessing/prepare_data.py
```

### Agent Workflow
Runs the CrewAI multi-agent system to classify tickets.
```powershell
python src/agents/ticket_crew.py
```

### Backend API
Starts the server for real-time classification and batch job management.
```powershell
python run_backend.py
```
*Access the interactive docs at: http://localhost:8000/docs*

### Testing
Runs the API tests and integration checks (Ensure API is running first).
```powershell
python -m pytest tests/
```
